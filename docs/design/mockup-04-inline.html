Hybrid search combines BM25's lexical matching with a vector reranking stage. BM25 handles rare-term recall and exact-match intent; the cross-encoder reranker then re-scores the top-K candidates for semantic relevance. The trick is in the candidate set — BM25 over the full corpus, then the reranker narrows to 20-50 chunks the LLM actually reads.
