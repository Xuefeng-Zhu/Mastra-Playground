Mastra Playground

Four small examples exercising Agent, Workflow, Tool, parallel execution, structured output, and live tracing.

Setup: cp .env.example .env, add OPENAI_API_KEY, then npm run serve. Open this page at http://localhost:8917.

Support Triage

Classify a customer message, then branch by intent. Watch the LLM call fire classify, then the framework test each branch predicate in order until one matches.

Human-in-the-Loop Approval

Submit a proposed action. The workflow classifies it with an LLM, then the gate step decides whether to auto-approve (low risk) or suspend for human review. High-urgency or large-amount actions always suspend. Click Approve or Reject to resume the workflow.

Workflow trace
Submit an action to see the workflow run.

Streaming Chat

Watch the LLM generate tokens one at a time. This is the same Agent as the other examples, but uses stream() instead of generate(). The UI shows each token as it arrives.

Workflow trace
Submit a prompt to watch the LLM stream.

Critic Loop

The evaluator-optimizer pattern. A generator drafts an answer, a critic scores it 0-10, and the loop regenerates using the feedback until the score meets the threshold (default 7) or the iteration budget runs out (default 3). Watch the score climb across iterations in the trace.

Workflow trace
Submit a topic to run the generator-critic loop.

Multi-Agent Handoff

A triage agent handles the customer's first message. If the question is about billing, it calls a tool that hands off to a specialist agent with a narrower prompt and focused tools. Watch the trace for the specialist step firing only when delegation occurs.
