Mastra Playground

Four small examples exercising Agent, Workflow, Tool, parallel execution, structured output, and live tracing.

Setup: cp .env.example .env, add OPENAI_API_KEY, then npm run serve. Open this page at http://localhost:8917.

Support Triage

Classify a customer message, then branch by intent. Watch the LLM call fire classify, then the framework test each branch predicate in order until one matches.

Human-in-the-Loop Approval

Submit a proposed action. The workflow classifies it with an LLM, then the gate step decides whether to auto-approve (low risk) or suspend for human review. High-urgency or large-amount actions always suspend. Click Approve or Reject to resume the workflow.
