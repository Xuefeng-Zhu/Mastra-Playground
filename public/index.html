Critic Loop

The evaluator-optimizer pattern. A generator drafts an answer, a critic scores it 0-10, and the loop regenerates using the feedback until the score meets the threshold (default 7) or the iteration budget runs out (default 3). Watch the score climb across iterations in the trace.

Mastra Memory

Uses the real @mastra/memory Memory class with @mastra/core/storage's InMemoryStore. The same threadId across two turns → second turn recalls what was said in the first. A different threadId → fresh conversation.

Content Pipeline

Three agents run in sequence as separate workflow steps: researcher gathers facts and sources, writer drafts ~150 words, and editor polishes and scores 0-10. Each agent has a narrow role prompt instead of one generalist doing everything. Compare to 02-research-agent where one agent does it all.
